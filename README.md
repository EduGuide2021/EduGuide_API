# EduGuide Server Using Laravel 7

# Dependencies Requirements
1. `php`
2. `composer`
3. `mysql`
4. `apache2`
5.


# Installation Guide
 1. ` cd edu_guide_server`
 2. `composer update`
 3. ` composer install`
 4. `php artisan serve (running localhost)`
 5. `php artisan serve --host ipaddr  --port number  (for any other machine can communicate)`
